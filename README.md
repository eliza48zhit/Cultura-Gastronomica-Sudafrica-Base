🛠️ Especificaciones Tecnicas del ADN Sudafricano
Este contrato ha sido purgado de caracteres especiales para maxima compatibilidad y valida la interaccion entre el metodo de calor y el tiempo de maduracion:

Tipo de Combustible (string): Clasifica la fuente de energia (Madera, Carbon o Aire). La seleccion de la madera (como el Kameeldoring) es critica para el perfil de sabor ahumado.

Dias de Curado (uint256): Variable que mide el tiempo de deshidratacion. Validado hasta un maximo de 30 dias para asegurar estandares de seguridad alimentaria on-chain.

Proteina de Caza (bool): Identificador booleano para distinguir el uso de fauna silvestre (Kudu, Avestruz, Impala) frente a ganaderia tradicional.

Sistema de Feedback: Funciones de darLike y darDislike para la interaccion de la comunidad.
